# Privacy Policy / プライバシーポリシー

Last updated: February 2, 2026

## English

### Overview
GitHub Roadmap Weekend & Holiday Highlighter ("the Extension") is committed to protecting your privacy. This privacy policy explains how the Extension handles your data.

### Data Collection
**The Extension does not collect, store, or transmit any personal data.**

Specifically, the Extension does NOT collect:
- Personal identification information
- Browsing history
- User activity or behavior data
- Location information
- Financial information
- Authentication credentials
- Any other personal data

### Data Storage
The Extension stores only the following data locally on your device:
- **Holiday data cache**: A list of Japanese national holidays fetched from the public Holidays JP API (https://holidays-jp.github.io/api/). This data is cached in your browser's local storage for 24 hours to improve performance. This cache contains only public holiday dates and names, not any personal information.

### Third-Party Services
The Extension connects to the following third-party service:
- **Holidays JP API** (https://holidays-jp.github.io/api/): Used to retrieve Japanese national holiday data. This is a public API that does not require authentication and does not receive any user data from the Extension.

### Permissions
The Extension requires the following permissions:
- **Access to github.com**: To run content scripts that highlight weekends and holidays on GitHub Projects roadmap view.
- **Access to holidays-jp.github.io**: To fetch public holiday data from the Holidays JP API.

### Changes to This Policy
We may update this privacy policy from time to time. Any changes will be posted on this page.

### Contact
If you have any questions about this privacy policy, please open an issue on the GitHub repository:
https://github.com/shiramizu-junya/github-roadmap-holidays/issues

## 日本語

### 概要
GitHub Roadmap Weekend & Holiday Highlighter（以下「本拡張機能」）は、ユーザーのプライバシーを保護することを約束します。このプライバシーポリシーは、本拡張機能がデータをどのように扱うかを説明します。

### データ収集
**本拡張機能は、個人データを一切収集、保存、送信しません。**

具体的に、本拡張機能は以下のデータを収集しません：
- 個人を特定できる情報
- 閲覧履歴
- ユーザーの活動や行動データ
- 位置情報
- 財務情報
- 認証情報
- その他の個人データ

### データ保存
本拡張機能は、以下のデータのみをお使いのデバイスにローカル保存します：
- **祝日データキャッシュ**: 公開されているHolidays JP API（https://holidays-jp.github.io/api/）から取得した日本の祝日リスト。このデータはパフォーマンス向上のため、ブラウザのローカルストレージに24時間キャッシュされます。このキャッシュには公開されている祝日の日付と名前のみが含まれ、個人情報は含まれません。

### サードパーティサービス
本拡張機能は以下のサードパーティサービスに接続します：
- **Holidays JP API**（https://holidays-jp.github.io/api/）: 日本の祝日データを取得するために使用します。これは認証を必要としない公開APIであり、本拡張機能からユーザーデータを受け取ることはありません。

### 権限
本拡張機能は以下の権限を必要とします：
- **github.comへのアクセス**: GitHub Projectsのロードマップビューで土日祝日をハイライト表示するコンテンツスクリプトを実行するために必要です。
- **holidays-jp.github.ioへのアクセス**: Holidays JP APIから祝日データを取得するために必要です。

### ポリシーの変更
このプライバシーポリシーは随時更新される場合があります。変更があった場合は、このページに掲載されます。
